# pawn_book
Flutter project-book rent management

## Project Setup

-  Clone the project
    ```bash
    git clone https://github.com/isajaisyullah/STQA-2A.git
    ```

-   Go to the new project folder.
    ```bash
    cd {new-project-folder}
    ```

-   Flutter get packages and run
    ```bash
    flutter pub get
    ```
    
    ```bash
    flutter build apk
    ```
    ```bash
    flutter run
    ```
    
    or just open it in VSCode
